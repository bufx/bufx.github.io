oracle 存储过程执行java进行密码加密成LDAP方式

步骤

1.先写 java
2.把相关包拷贝到当前目录下（rt.jar）
3.查询 oracle 的 jdk 版本
4.javac 执行编译（要用 oracle 自带的 jdk，因为版本不一致的话 loadjava 报错）
5.loadjava 导入到 oracle 中
6.sql 查询是否存在 Java
7.写存储过程
8.执行存储过程（两种方法）
8.1 直接查询
8.2 sqlplus 执行
9.删除 java 类
